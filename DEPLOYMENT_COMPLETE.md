## 🚀 Quick Setup for New Users:

### Windows Users:
```batch
git clone https://github.com/ivocreates/messMenu.git
cd messMenu
setup_windows.bat
start_app_windows.bat
```

### Linux/Mac Users:
```bash
git clone https://github.com/ivocreates/messMenu.git
cd messMenu
chmod +x setup.sh start_app.sh
./setup.sh
./start_app.sh
```
